## Install
建议使用`UV`进行环境配置，详情参考：[install uv](https://docs.astral.sh/uv/getting-started/installation/)
也可以直接使用conda安装：
```bash
conda install uv
```

```bash
git clone https://github.com/Aye10032/llm-annotation
cd llm-annotation
uv venv
# 根据提示激活环境
uv sync
```
